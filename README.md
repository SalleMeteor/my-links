my-links    --  Jordi Grau 2014
========

http://mylinkssalle.meteor.com

=====================================

My-links es una petita web on els usuaris poden afegir enllaços d'internet que 
considerin interessants.

Cada usuari te un login i un password que pot utilitzar per a pujar els enllaços 
Mylinks també permet que els usuaris votin positivament o negativament els enllaços.
L'aplicaicó mostrarà els enllaços ordenats de millor a pitjor puntuació.

Per acabar, cada usuari pot deixar els seus comentaris sobre els enllaços publicats.

------------------------------------------------------------------------------------
My links utulitza els seguents paquets:


      1-Iron-router: per fer la pestanya "about".
      2-Meteor-profile: per les comptes d'usuari.
      3-boostrap-3: front-end framework per la correcta visualització en mòbils.
      4-blaze-layout: Altres paquets el necessitaven pel correcte funcionament.
